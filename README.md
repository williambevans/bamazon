# Bamazon-Node-APP

Amazon-like storefront with the MySQL and Node. The app will take in orders from customers and deplete stock from the store's inventory

# Technologies Used

* Node.js
* JavaScript
* MySQL

# Dependencies

* Inquirer
  * npm install inquirer
* Figlet
  * npm install figlet
